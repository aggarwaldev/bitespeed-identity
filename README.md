### How to install DB Driver
From [typeorm docs](https://typeorm.io/#installation):

- for MySQL or MariaDB

`yarn add mysql2`

- for PostgreSQL or CockroachDB

`yarn add pg`

- for SQLite

`yarn add sqlite3`

- for Microsoft SQL Server

`yarn add mssql`

- for sql.js

`yarn add sql.js`
